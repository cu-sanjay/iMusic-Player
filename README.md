# YouTube to MP3 Converter

A minimal and optimized YouTube to MP3 converter web application.

## Features

- Convert YouTube videos to MP3
- Mobile-friendly UI
- Error handling for all scenarios
- Fast and responsive
- Minimal codebase

## Technologies

- Next.js
- React
- Tailwind CSS
- ytdl-core

## Deployment

Deployed on Vercel: [https://utubemusic.vercel.app/](https://utubemusic.vercel.app/)
