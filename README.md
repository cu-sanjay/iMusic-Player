# UtubeMusic - Video to Audio Downloader

![UtubeMusic Logo](/public/images/UTUBE%20White.png)

UtubeMusic is a sleek, user-friendly tool to extract audio from video links and download it as MP3 files. Perfect for enjoying your favorite tracks offline!

## Features
- **Fast & Simple**: Convert videos to MP3 in just a few clicks.
- **No Installation**: Runs directly in your browser.
- **Open Source**: Built by [Sannjay](https://github.com/cu-sanjay/).

## How to Use
1. Visit the UtubeMusic website (deployed on Vercel).
2. Paste a video URL in the input field.
3. Click "Get Audio Now" and wait 20-30 seconds.
4. Download your MP3 file and enjoy!

## Installation
No setup required! Just head to the deployed site and start converting. For developers:
1. Clone the repo: `git clone https://github.com/cu-sanjay/MyYoutube`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`

## Tech Stack
- **Backend**: Node.js, Express
- **Frontend**: EJS, CSS
- **Audio Extraction**: yt-dlp
- **Deployment**: Vercel
